## Available Scripts

In the project directory, you can run:

### `npm run build-main`

打包主进程

### `npm run build-renderer`

打包渲染进程

### `npm run build`

主进程和渲染进程并行打包

### `npm run start-main`

运行 Electron 项目

### `npm run dev`

开发环境运行 Electron 项目

### `npm run build && npm run start-main`

生产环节打包并运行 Electron 项目
